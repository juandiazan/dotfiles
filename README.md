# config-files

## Setup Guide

Store git credentials:

```bash
git config --global credential.helper store
```
Change shell to zsh (after install):

```bash
chsh -s $(which zsh)
```

Install omz:
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
